## HSPA (3.5G)

### HSDPA (High Speed Downlink Packet Access)

| Feature | Value |
|---------|-------|
| Peak speed | 14.4 Mbps (Cat 10) |
| TTI | 2 ms (vs 10-20 ms) |
| Modulation | QPSK, 16-QAM |
| Channels | HS-DSCH (shared) |

### HSUPA (High Speed Uplink Packet Access)

| Feature | Value |
|---------|-------|
| Peak speed | 5.76 Mbps |
| TTI | 2 ms / 10 ms |
| Modulation | BPSK, QPSK |
| Channels | E-DCH |

### HSPA+ (3.75G)

| Feature | Value |
|---------|-------|
| Peak DL | 42 Mbps (DC-HSPA+) |
| Peak UL | 11.5 Mbps |
| Modulation | 64-QAM |
| MIMO | 2x2 optional |

## Channels

### Logical Channels

| Channel | Direction | Purpose |
|---------|-----------|---------|
| BCCH | DL | Broadcast system info |
| PCCH | DL | Paging |
| CCCH | Both | Common control |
| DCCH | Both | Dedicated control |
| DTCH | Both | Dedicated traffic |

### Transport Channels

| Channel | Purpose |
|---------|---------|
| BCH | Broadcast |
| PCH | Paging |
| FACH | Forward access |
| RACH | Random access |
| DCH | Dedicated |
| HS-DSCH | HSDPA data |
| E-DCH | HSUPA data |

## Handover Types

| Type | Description |
|------|-------------|
| Soft | Connected to multiple cells simultaneously |
| Softer | Same Node B, different sectors |
| Hard | Break-before-make (inter-frequency, inter-RAT) |
