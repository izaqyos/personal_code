### Key Components

| Component | Function |
|-----------|----------|
| **SGSN** | Serving GPRS Support Node - mobility, auth, routing |
| **GGSN** | Gateway GPRS Support Node - IP gateway to internet |
| **PCU** | Packet Control Unit - packet scheduling at BSC |

### GPRS Classes

| Class | Operation | Use Case |
|-------|-----------|----------|
| Class A | Simultaneous voice + data | Rare |
| Class B | Automatic switching | Common phones |
| Class C | Manual switching | Data-only devices |

### Coding Schemes

| Scheme | Data Rate | Robustness |
|--------|-----------|------------|
| CS-1 | 9.05 kbps | Most robust |
| CS-2 | 13.4 kbps | Good |
| CS-3 | 15.6 kbps | Medium |
| CS-4 | 21.4 kbps | Least robust |

### Theoretical Max Speed

```
Max slots: 8
Max per slot: 21.4 kbps (CS-4)
Total: 8 Ã— 21.4 = 171.2 kbps (theoretical)
Real-world: 30-80 kbps typical
```
