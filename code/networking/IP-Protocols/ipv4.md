## Header Fields

| Field | Size | Description |
|-------|------|-------------|
| Version | 4 bits | Always 4 for IPv4 |
| IHL | 4 bits | Header length in 32-bit words (min 5 = 20 bytes) |
| DSCP | 6 bits | Differentiated Services Code Point (QoS) |
| ECN | 2 bits | Explicit Congestion Notification |
| Total Length | 16 bits | Entire packet size (max 65,535 bytes) |
| Identification | 16 bits | Fragment identification |
| Flags | 3 bits | DF (Don't Fragment), MF (More Fragments) |
| Fragment Offset | 13 bits | Position in fragmented packet |
| TTL | 8 bits | Time to Live (hop count) |
| Protocol | 8 bits | Upper layer protocol (6=TCP, 17=UDP) |
| Checksum | 16 bits | Header checksum |
| Source IP | 32 bits | Sender's IP address |
| Destination IP | 32 bits | Recipient's IP address |
