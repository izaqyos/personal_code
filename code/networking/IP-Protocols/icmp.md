## ICMPv6

ICMPv6 (Protocol 58) is more important than ICMPv4:

| Type | Name | Description |
|------|------|-------------|
| 1 | Dest Unreachable | Like ICMPv4 Type 3 |
| 2 | Packet Too Big | Like ICMP "Frag Needed" |
| 3 | Time Exceeded | TTL expired |
| 4 | Parameter Problem | Bad header |
| 128 | Echo Request | Ping |
| 129 | Echo Reply | Ping reply |
| 133 | Router Solicitation | Find routers (NDP) |
| 134 | Router Advertisement | Router info (NDP) |
| 135 | Neighbor Solicitation | Find MAC (NDP, like ARP) |
| 136 | Neighbor Advertisement | Reply with MAC (NDP) |
| 137 | Redirect | Better route |

```
ICMPv6 is CRITICAL for IPv6:
â€¢ No ARP - uses Neighbor Discovery (NS/NA)
â€¢ SLAAC depends on RA messages
â€¢ Don't block ICMPv6!
```

## Tools

```bash
# Basic ping
ping 8.8.8.8
ping -c 4 8.8.8.8           # 4 packets
ping -i 0.5 8.8.8.8         # 0.5 sec interval
ping -s 1000 8.8.8.8        # 1000 byte payload

# Ping with specific TTL
ping -t 10 8.8.8.8          # TTL=10 (Windows)
ping -m 10 8.8.8.8          # TTL=10 (Linux/Mac)

# Traceroute
traceroute 8.8.8.8
traceroute -I 8.8.8.8       # Use ICMP instead of UDP
traceroute -n 8.8.8.8       # No DNS resolution

# MTU discovery
ping -M do -s 1472 8.8.8.8  # DF set, find MTU

# Capture ICMP
tcpdump -i eth0 icmp
tcpdump -i eth0 'icmp[icmptype] == 8'  # Echo requests only

# Wireshark filter
icmp
icmp.type == 8
icmp.type == 3 && icmp.code == 4
```
