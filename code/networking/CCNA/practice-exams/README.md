## How to Use

### First Attempt
1. Set a timer for 60 minutes
2. No notes or references
3. Mark uncertain questions
4. Complete all questions before checking

### Review
1. Check answers
2. Research wrong answers
3. Note weak areas in PROGRESS.md
4. Review relevant module content

### Retake
1. Wait at least 1 week
2. Retake the exam
3. Track improvement
