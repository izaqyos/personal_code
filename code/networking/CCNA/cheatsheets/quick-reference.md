## OSPF Quick Reference

### Network Types
| Type | DR/BDR | Hello/Dead |
|------|--------|------------|
| Broadcast | Yes | 10/40 |
| Point-to-Point | No | 10/40 |
| NBMA | Yes | 30/120 |

### LSA Types
| Type | Name | Generated By |
|------|------|-------------|
| 1 | Router | Every router |
| 2 | Network | DR |
| 3 | Summary | ABR |
| 4 | ASBR Summary | ABR |
| 5 | External | ASBR |
| 7 | NSSA External | ASBR in NSSA |

### OSPF Cost
```
Cost = Reference BW / Interface BW
Default reference = 100 Mbps

Recommended: auto-cost reference-bandwidth 10000
```

## STP Quick Reference

### Port States
| State | Learn | Forward |
|-------|-------|---------|
| Blocking | No | No |
| Listening | No | No |
| Learning | Yes | No |
| Forwarding | Yes | Yes |

### Port Roles
- **Root Port**: Best path to root bridge
- **Designated Port**: Best port per segment
- **Alternate/Blocked**: Backup paths

### STP Timers
| Timer | Default |
|-------|---------|
| Hello | 2 sec |
| Forward Delay | 15 sec |
| Max Age | 20 sec |

## VLAN Commands

```cisco
! Create VLAN
vlan 10
name SALES

! Access port
switchport mode access
switchport access vlan 10

! Trunk port
switchport mode trunk
switchport trunk allowed vlan 10,20,30
switchport trunk native vlan 99

! Verification
show vlan brief
show interfaces trunk
```

---

## Routing Commands

```cisco
! Static route
ip route 192.168.2.0 255.255.255.0 10.1.1.2

! Default route
ip route 0.0.0.0 0.0.0.0 10.1.1.1

! OSPF
router ospf 1
router-id 1.1.1.1
network 10.0.0.0 0.255.255.255 area 0
passive-interface gi0/0

! Verification
show ip route
show ip ospf neighbor
show ip ospf interface brief
```

## Security Commands

```cisco
! Port security
switchport port-security
switchport port-security maximum 2
switchport port-security mac-address sticky
switchport port-security violation shutdown

! DHCP snooping
ip dhcp snooping
ip dhcp snooping vlan 10
ip dhcp snooping trust (on trusted port)

! ACL
access-list 100 permit tcp any any eq 443
ip access-group 100 in
```
