## Tips for Effective Practice

1. **Time yourself** - Build speed with CLI drills
2. **Don't peek** - Try to solve before checking answers
3. **Break things** - Intentionally misconfigure, then fix
4. **Document** - Write down what you learn
5. **Repeat** - Spaced repetition for retention
