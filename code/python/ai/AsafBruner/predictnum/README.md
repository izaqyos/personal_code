# Number Prediction

This project is about predicting numbers using a trained Artifical Neural Network model.

## Prerequisites

Before you begin, ensure you have met the following requirements:

* All the required Python libraries are installed in your computer

```bash
pip install -r requirements.txt
```

* You have a `mnist_model.weights.h5` file which is a trained model. If you don't have it, you can train the model by running the following command:

```bash
python ./modelutils.py train
